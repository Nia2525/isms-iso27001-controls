# Short Explanation

This repository is a lab-style ISMS portfolio focused on a small but meaningful subset of ISO/IEC 27001:2022 controls (A.8.6–A.8.10).

Each control file answers the practical questions:

- What is this control trying to achieve (objective)?  
- How could an organization implement it (implementation guidance)?  
- What would an auditor look for (audit criteria and evidence)?  
- What problems are commonly found (typical findings)?  
- How mature is the implementation (maturity indicators)?  

The goal is to make abstract standards tangible and to show how a security professional can think in terms of controls, governance, and audit readiness.
