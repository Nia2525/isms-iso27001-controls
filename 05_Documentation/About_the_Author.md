# About the Author

This lab portfolio was created by Kurnia Wijayanti as part of a practical exploration of ISO/IEC 27001:2022 controls and Information Security Management Systems (ISMS).

Kurnia combines hands-on experience in security operations with a growing focus on governance, risk management, and compliance. This project is intended to demonstrate:

- Structured thinking about controls and their objectives  
- Audit-oriented documentation style  
- Ability to translate technical measures into governance language  

The repository is designed as a living portfolio that can evolve with further controls, scenarios, and evidence examples.
